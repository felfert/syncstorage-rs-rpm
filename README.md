# syncstorage-rs-rpm

This is the source for packaging [Mozilla's syncstorage-rs syncserver](https://github.com/mozilla-services/syncstorage-rs) on [COPR](https://copr.fedorainfracloud.org/coprs/felfert/syncstorage-rs/) for Fedora and RHEL and on [Launchpad](https://launchpad.net/~felfert/+archive/ubuntu/syncstorage-rs) for Ubuntu.
